# Zen (Browser)
Download [Brew](https://github.com/BosEriko/brew) as your package manager then install [Zen](https://formulae.brew.sh/cask/zen).

## Install Zen
```sh
brew install --cask zen
```

## Clone the Repository
```sh
mkdir -p ~/Documents/Codes/Configuration
git clone https://github.com/BosEriko/zen.git ~/Documents/Codes/Configuration/zen
```

## Update your settings
Follow this [instruction](instruction.md) to update your settings. Then update your [hotkeys](hotkeys.md).
