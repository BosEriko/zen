# Hotkeys
Add the hotkeys [here](brave://settings/system/shortcuts).

## Compact Mode
- Toggle Floating Sidebar: `ctrl`+`\`
## Other Zen Features
- Copy current URL: `ctrl`+`b`
## Window & Tab Management
- New Tab: `ctrl`+`n`
- Close Tab: `ctrl`+`w`
- Restore Last Closed Tab: `ctrl`+`shift`+`n`
## Navigation
- Go Back: `ctrl`+`h`
- Go Forward: `ctrl`+`l`
- Reload Page: `ctrl`+`r`
- Reload Page (Skip Cache): `ctrl`+`shift`+`r`

# Unavailable
- Next Tab: `ctrl`+`j`
- Previous Tab: `ctrl`+`k`
